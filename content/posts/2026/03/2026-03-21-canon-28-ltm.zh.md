不挤牙膏的佳能什么样？
<!--more-->

# 痒

最近沉迷小巧精致的镜头。本想收一枚福伦达新出的28 2.8，奈何看了样片发现是高反差偏冷色的风格（跟之前出掉的VM 35 1.5很像），不喜。然后发现福伦达千禧年间生产过螺口版的28 3.5 LTM似乎不错，可惜那枚产量稀少溢价严重。又看了看类似体积的银色饼干头：阿部光学Avenon 28 3.5 LTM - 白色铭文环造型不喜；美能达G-Rokkor 28 3.5 LTM - 白点起雾隐患；尼康W-Nikkor 28 3.5 LTM - 光圈环操作不便；理光GR 28 2.8 LTM - 很好但贵。最后把目光落回半个世纪前，收了枚佳能1957年发布的28 2.8 LTM。

{{< img src="/images/2026/03/21/m3-28-vf.JPG" title="↑佳能28 2.8 LTM" caption="搭配1959年产的徕卡M3">}}

# 史

跟上世纪中叶的[尼康螺口镜头]({{< ref "/posts/2021/11/2021-11-13-m9-nikkor.zh.md" >}})类似，这枚28 2.8属于佳能早期为了开拓旁轴市场铆足劲下血本打造的匠心之作，从1957年一直生产到了1975年。随机一起入手了原装皮套和原厂28mm取景器，也都做工精湛（可惜取景器无框线）。其实早在1951年，佳能就发布过光圈更小的Serenar 28 3.5 LTM：这是佳能历史上第一枚28mm镜头，同样是全银色镜身；1957年起，估计是因为同年发布了这枚规格更高的2.8版本，3.5版的镜桶造型改成了黑白色，体积也变大（丑）了。看来佳能早在半个世纪前就懂得使用精湛刀法减配物料来区分产品线了。

{{< img src="/images/2026/03/21/coating.JPG" title="↑佳能28 2.8 LTM" caption="洋红色镀膜">}}

# 析

{{< img src="/images/2026/03/21/lens-group.jpg" title="↑镜头宣传册" caption="可以看到半对称的6片4组镜片结构">}}

镜头长度仅20mm，重159克，用料扎实（作为对比，福伦达VM 28 2.8 type ii是23.5mm，106克），6片4组类Topogon又带点Biogon影响的半对称结构。全开光圈时中心锐度很不错，畸变也很小，但类似Topogon的结构导致出瞳光线倾斜角度很大，如果接在M9数码机身上，边角画质会有明显衰退伴随巨大暗角（不过还好没有明显红移偏色）；如果拍胶片，边角会好看很多。颜色偏暖反差中等，非常古典的味道。值得一提的是，著名街拍摄影师Garry Winogrand就用这枚头拍摄了不少早期素材。

{{< img src="/images/2026/03/21/m3-28-side.JPG" title="↑搭配M3" caption="无28mm框线，需要外置取景器">}}

{{< img src="/images/2026/03/21/m9-28.JPG" title="↑搭配M9" caption="可以使用M9取景器自带的28mm框，不过眼镜党看着会比较费劲">}}

# 转

如果将这枚镜头转接在徕卡CL无反机身上，APSC画幅裁切正好规避了崩边和暗角，等效42mm也很适合挂机（40mm爱好者狂喜）。

{{< img src="/images/2026/03/21/cl-28.JPG" title="↑搭配CL无反" caption="非常趁手的搭配，后面是类似焦段的禄来40 2.8 LTM和宾得大公主FA 43 1.9">}}

{{< img src="/images/2026/03/21/canon-28-rollei-40.JPG" title="↑Canon 28 2.8 LTM" caption="这枚比禄莱40 2.8 LTM还小一点">}}

# 样

小巧镜头随身携带一整天也毫无负担，大大增加了平时带出门的概率。光线好的时候，把光圈调到f8，从1.5米到无穷远都在景深内，不管对焦直接取景拍就好，超焦距拍摄的体验就是爽快。以下是样片。
